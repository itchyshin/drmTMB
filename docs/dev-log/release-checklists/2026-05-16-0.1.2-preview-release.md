## Future Release Rule

Every tagged preview should run the install smoke after the tag is pushed:

```sh
Rscript tools/install-smoke.R vX.Y.Z X.Y.Z
```

This catches mismatches between the tag users install and the documentation they
read on the current pkgdown site.
