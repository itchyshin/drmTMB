# Release Checklist: drmTMB 0.1.2 Preview

This checklist records the `0.1.2` GitHub preview release gate after the Phase
6 profile-inference hardening, Phase 6e tutorial maturation, and Phase 17-20
roadmap reorder.

Labels: `release`, `0.1.2`, `pkgdown`, `install`

## Current Evidence Snapshot

- `v0.1.1` remains immutable and installable as the corrected storage-control
  preview.
- `v0.1.2` is intended as the next tagged preview for users who need the
  profile-inference hardening, stable-core validation evidence, and refreshed
  Gaussian plus structural-dependence tutorials.
- The active install instructions should point to
  `pak::pak("itchyshin/drmTMB@v0.1.2")`.
- The release is a GitHub preview tag, not a CRAN submission.

## Future Release Rule

Every tagged preview should run the install smoke after the tag is pushed:

```sh
Rscript tools/install-smoke.R vX.Y.Z X.Y.Z
```

This catches mismatches between the tag users install and the documentation they
read on the current pkgdown site.
