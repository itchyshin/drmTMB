# Release Checklist: drmTMB 0.1.1 Preview

This checklist records the patch preview that corrected the released install
target after `v0.1.0` was found not to include the exported `drm_control()`
storage controls.

Labels: `release`, `0.1.1`, `pkgdown`, `install`

## Current Evidence Snapshot

- `v0.1.0` remains immutable and installable, but it does not export
  `drm_control()`.
- `v0.1.1` is the corrected tagged preview for users who need
  `drm_control(keep_data = FALSE, keep_model_frame = FALSE,
  keep_tmb_object = FALSE)`.
- Main-branch GitHub Actions `R-CMD-check` run `25639248630` passed on macOS,
  Ubuntu, and Windows for commit `b4e222c`.
- Tag-triggered GitHub Actions `R-CMD-check` run `25639416001` passed on macOS,
  Ubuntu, and Windows for tag `v0.1.1`.
- GitHub Actions `pkgdown` run `25639387716` deployed the site for commit
  `b4e222c`.
- `Rscript tools/install-smoke.R v0.1.1 0.1.1` installed the tag into a clean
  temporary library, confirmed package version `0.1.1`, confirmed the exported
  functions `drmTMB`, `bf`, `drm_control`, and `check_drm`, and fitted a
  storage-control Gaussian location-scale smoke model.

## Future Release Rule

Every tagged preview should run the install smoke after the tag is pushed:

```sh
Rscript tools/install-smoke.R vX.Y.Z X.Y.Z
```

This catches mismatches between the tag users install and the documentation they
read on the current pkgdown site.
