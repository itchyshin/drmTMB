# Release Checklist: drmTMB 0.1.3 Preview

This checklist records the `0.1.3` GitHub preview release gate after the first
known-relatedness Gaussian `mu` slice and the first coordinate-spatial bivariate
q=2 location-covariance slice.

Labels: `release`, `0.1.3`, `pkgdown`, `install`

  and tutorial-hardening preview.

  fitted coordinate-spatial bivariate `mu1`/`mu2` q=2 covariance slice.

## Future Release Rule

Every tagged preview should run the install smoke after the tag is pushed:

```sh
Rscript tools/install-smoke.R vX.Y.Z X.Y.Z
```

This catches mismatches between the tag users install and the documentation they
read on the current pkgdown site.
