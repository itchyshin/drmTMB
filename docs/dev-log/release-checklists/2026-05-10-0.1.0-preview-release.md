# Release Checklist: drmTMB 0.1.0 Preview

This is an issue-ready checklist for the first public preview release. Version
`0.1.0` should mean a reliable, clearly bounded package that users can try; it
should not claim full double-hierarchical individual-difference covariance or
every planned family.

Suggested labels: `release`, `0.1.0`, `pkgdown`, `CRAN-ish`.

Tracking issue: https://github.com/itchyshin/drmTMB/issues/1

## Current Evidence Snapshot

This snapshot records release-gate evidence gathered on 2026-05-10. It is not
the final release sign-off.

- GitHub issue #1 now tracks this checklist publicly.
- `Rscript tools/replicate-location-scale-gaussian.R` passed. The fixed-effect
  scenario had maximum absolute differences of `1.37e-06` for `mu`
  coefficients, `2.00e-06` for `sigma` coefficients, and `3.96e-10` for
  log-likelihood. The random-intercept scenario had maximum absolute
  differences of `6.23e-08` for `mu` coefficients, `6.68e-06` for `sigma`
  coefficients, `6.81e-07` for the `mu` random-intercept SD, and `2.12e-09`
  for log-likelihood.
- The numeric comparator result is archived in
  `docs/dev-log/comparator-results/2026-05-10-gaussian-location-scale-glmmtmb.md`.
- `Rscript -e "devtools::document()"` ran after the comparator check and
  produced no file changes.

## Scope For 0.1.0

Include the implemented one-response and two-response workflows that already
have tests and user-facing documentation:

- Gaussian location-scale models with the implemented random-effect paths.
- Gaussian known sampling variance or covariance through `meta_known_V(V = V)`.
- Fixed-effect bivariate Gaussian location-coscale models with residual
  `rho12`.
- Implemented positive-continuous, count, proportion, beta-binomial, and
  cumulative-logit ordinal MVPs.
- Implemented extractors, checks, fitted values, simulation, and reference
  articles.

Keep these as post-0.1.0 roadmap work unless they are implemented and tested
before the release:

- Full double-hierarchical covariance among personality, plasticity,
  predictability, and malleability.
- Bivariate random-effect covariance blocks and non-residual correlation-pair
  extensions.
- Ordinal scale/discrimination formulae.
- Denominator response aliases beyond `cbind(successes, failures)`.
- Routine spatial fitting and phylogenetic location-scale extensions.
