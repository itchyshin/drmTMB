# Figure Audit: Reference Plot Helpers

## Scope

This pass inspected the rendered images generated by the `plot_parameter_surface()`
and `plot_corpairs()` reference examples. These are small helper examples, but
they appear on public reference pages and should not teach users with distorted
or visually empty fixtures.

Render command:

```sh
Rscript -e "pkgdown::build_reference()"
```

Active rendered images:

```sh
find pkgdown-site/reference -maxdepth 1 \( -name 'plot_*.png' -o -name '*corpairs*.png' -o -name '*surface*.png' \) -print -exec file {} \;
```

## Remaining Watch Items

- These are reference examples, not exported theme decisions. The broader
  visualization layer still belongs to #58.
- `plot_parameter_surface()` and `plot_corpairs()` intentionally remain small
  `ggplot2` consumers. More polished article figures can add themes after the
  helper returns a `ggplot` object.
