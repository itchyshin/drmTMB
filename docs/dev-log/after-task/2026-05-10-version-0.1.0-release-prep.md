# After Task: Version 0.1.0 Release Preparation

## Goal

Move `drmTMB` from a development-version preview target to a tagged `0.1.0`
preview candidate that users can install from GitHub.

## Implemented

`DESCRIPTION` now reports version `0.1.0`, `NEWS.md` has the dated heading
`drmTMB 0.1.0 (2026-05-10)`, the landing page describes this as the first
public preview, the pkgdown badge says `0.1.0 preview release`, and the roadmap
now describes the `0.1.0` preview boundary rather than a future release gate.

## Mathematical Contract

No model, likelihood, extractor, or formula grammar changed. The release keeps
the same public scope: implemented one-response and two-response workflows are
available, public `sigma` remains the modelled scale parameter, `rho12` remains
the residual bivariate correlation parameter, and full double-hierarchical
covariance, bivariate random-effect covariance, ordinal scale formulae,
denominator aliases, spatial effects, phylogenetic scale effects, skew-normal,
and skew-t remain post-preview work.

to keep shell search patterns boring when checking release strings.

scripts, variance-facing `sigma^2` reporting helpers, and the planned
