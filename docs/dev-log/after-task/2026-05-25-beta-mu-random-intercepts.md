# After Task: Beta `mu` Random Intercepts

## Goal

Add the first `beta()` mixed-model slice: ordinary unlabelled `mu` random
intercepts with `bf(prop ~ x + (1 | id), sigma ~ z)` for strict `(0, 1)`
responses.

## Implemented

`beta()` now accepts ordinary unlabelled location random intercepts in the
`mu` formula. The random intercept enters the logit-`mu` predictor, the
response must stay strictly inside `(0, 1)`, and `sigma` remains a fixed-effect
log-scale formula.

The fitted random-effect SD is exposed through `sdpars$mu`, conditional effects
through `random_effects$mu`, direct interval targets through
`profile_targets()`, and first-pass replication diagnostics through
`check_drm()`.

## Mathematical Contract

For observation `i` in group `j`:

```text
logit(mu_ij) = X_mu[i, ] beta_mu + b_j
b_j = sd_mu u_j
u_j ~ Normal(0, 1)
log(sigma_i) = X_sigma[i, ] beta_sigma
phi_i = 1 / sigma_i^2
y_i ~ Beta(mu_i phi_i, (1 - mu_i) phi_i)
```

This contract does not open beta random slopes, beta `sigma` random effects,
labelled covariance blocks, exact 0/1 boundary mass, `zoi`/`coi`, structured
bounded-response effects, known covariance, beta-binomial denominator-aware
random effects, or bivariate/mixed bounded-response models.

## Tests Of The Tests

The new beta recovery test initially failed because the TMB data builder was
not passing `re_mu` into the beta branch. After that was fixed, it failed again
because `check_drm()` did not route beta through random-effect replication
diagnostics. Those failures showed that the test exercised both the likelihood
wiring and the public diagnostic surface, not only object construction.

The malformed-neighbour tests cover beta random slopes, labelled covariance,
`sigma` random effects, strict response boundaries, and planned `zoi`/`coi`
paths.

## What Did Not Go Smoothly

The first beta implementation built the random-effect structure in R but forgot
to pass it through `make_tmb_data()` for `model_type = 10`, which made the TMB
branch look fixed-effect only. The first broad test run also exposed a stale
shared scale-boundary expectation that assumed one generic non-Gaussian error
message. Finally, `air format .` touched unrelated files; those formatting-only
changes were trimmed out before closing the task.

## Known Limitations

This is a narrow first slice. The following remain planned or unsupported:
beta random slopes, beta `sigma` random effects, labelled covariance blocks,
random-effect scale formulae, `zoi`/`coi`, exact 0/1 boundary mass, structured
bounded-response effects, known covariance, beta-binomial denominator-aware
random effects, and bivariate or mixed bounded-response models.

## Next Actions

The natural next implementation lane is `beta_binomial()` `mu` random
intercepts, because it is related to beta `mu` random intercepts but needs
denominator/count handling and a separate recovery test.
