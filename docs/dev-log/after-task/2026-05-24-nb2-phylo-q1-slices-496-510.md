# After Task: NB2 Phylogenetic q1 Implementation, Slices 496-510

## Goal

Fit the ordinary, non-zero-inflated NB2 q=1 phylogenetic `mu` intercept:

```r
bf(count ~ x + phylo(1 | species, tree = tree), sigma ~ z)
```

The task keeps `sigma` as fixed-effect NB2 overdispersion and does not admit
NB2 `sigma` phylogeny or other structured scale effects, zero-inflated NB2
phylogeny, hurdle/inflation random effects, structured slopes, q2/q4 count
covariance, spatial, animal, or `relmat()` count structure. A later same-day
slice opens the separate ordinary NB2 log-`sigma` random-intercept gate.

## Implemented

`drm_build_nbinom2_spec()` now extracts one unlabelled
`phylo(1 | species, tree = tree)` term from ordinary NB2 `mu`, builds the
phylogenetic structured-effect object, passes the sparse precision path to TMB,
starts and maps `u_phylo` and `log_sd_phylo`, and marks `u_phylo` as a random
effect. The NB2 TMB branch adds the phylogenetic log-mean contribution before
evaluating the NB2 density and adds the same Gaussian sparse-precision prior
used by the Poisson q1 route.

The fitted route appears through `sdpars$mu`, `ranef("phylo_mu")`,
`profile_targets()` with direct target `sd:mu:phylo(1 | species)`, and
`check_drm()` phylogenetic diagnostics. Shared post-fit helpers now count
ordinary count `mu` random effects and NB2 structured `mu` effects correctly,
so `predict(fit, dpar = "mu")` includes the fitted random field for the new
route.

## Mathematical Contract

The fitted model is:

```text
y_i | a ~ NB2(mu_i, size_i = 1 / sigma_i^2)
log(mu_i) = offset_i + x_i beta_mu + a_species[i]
log(sigma_i) = z_i beta_sigma
a ~ Normal(0, sigma_phylo^2 K_phylo)
```

Here `sigma` is the public NB2 overdispersion scale, with
`Var(y_i | a) = mu_i + sigma_i^2 * mu_i^2`. The phylogenetic effect is a
structured random intercept on the log-mean predictor only. There is no q1
latent-correlation row for `corpairs()`.

to distinguish already-finished Poisson formal-admission work from the

with a realistic numerical tolerance exposed the real helper risk: prediction
and random-effect term counting needed to recognize count-model random effects
instead of only Gaussian ones.

## Known Limitations

No overdispersion-aware formal recovery grid has been run for NB2 q1. The route
is smoke-tested and extractor-tested, not broadly admitted for operating
characteristics.

Within this NB2 phylogenetic `mu` slice, `sigma` remains fixed-effect only. A
later same-day slice opens independent ordinary NB2 log-`sigma` random
intercepts; structured effects in `sigma`, `zi`, `hu`, future `coi`, structured
count slopes, q2/q4 count covariance, spatial, animal, `relmat()`,
zero-truncated NB2 random effects, and bivariate count models remain planned or
blocked.
