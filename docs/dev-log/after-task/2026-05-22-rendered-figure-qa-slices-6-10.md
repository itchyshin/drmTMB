## What Did Not Go Smoothly

The first `relmat()` q=2 render was point-only because the small simulated
example produced a one-sided profile interval with an unavailable lower bound.
The article now sets the simulation seed explicitly to a small example with
finite profile bounds for both q=2 structural-correlation rows.

## Team Learning

Fisher's rule is now sharper: a fitted q=2 correlation row should not be
point-only when direct profile intervals are available. Rose's companion rule
is just as important: do not force q=4 derived rows into Confidence Eyes before
the interval machinery supports that claim.

## Known Limitations

Reference example images generated by pkgdown do not currently carry the same
caption and alt-text hooks used by article figures. The figure content is
clearer, but a later accessibility pass should handle reference example image
alt text explicitly.
