# Beta phylogenetic q1 disjoint repair design

described in
`docs/dev-log/2026-07-16-beta-phylo-q1-pr1-disjoint-seed-repair-plan.md`.

`tools/run-beta-phylo-q1-recovery.R --mode=repair_design`. `seed-audit.tsv`
proves that all certification seeds are unique and that none occurs in the
original `m = 2` campaign, the invalid-overlap `m = 4` campaign, or the repair
smoke and pilot schedules.
